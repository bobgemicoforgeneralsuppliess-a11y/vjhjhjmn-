## 🎯 Executive Summary
The **E-Commerce Store Management & Analytics Ecosystem** is a professional-grade, full-scale data architecture project developed over 45 days. It spans the entire data lifecycle—from robust relational database design and complex SQL queries to advanced Exploratory Data Analysis (EDA), predictive Machine Learning modeling, and real-time Business Intelligence tracking. This project transforms raw e-commerce transactions into strategic, actionable business insights.

## ⚙️ End-to-End Workflow Overview (The A to Z)

### 🗄️ Phase 1: Data Engineering & Database Architecture (SQL Server)
The foundation of the ecosystem relies on a highly optimized, structured relational database designed to handle high-volume transactional data.
*   **Architecture:** 14 fully normalized tables (e.g., `Customers`, `Orders`, `Products`, `Invoices`) secured with 22 Foreign Key (FK) constraints to ensure absolute data integrity.
*   **ETL Processes:** Automated bulk data ingestion pipelines mapping raw CSVs into structured SQL schemas.
*   **Analytical Queries:** Developed complex SQL views and queries to answer **20 strategic business questions** (e.g., daily KPIs, top-ranking products, and delivery delay reporting).

### 🐍 Phase 2: Exploratory Data Analysis & Feature Engineering (Python)
*Led by Mohamed Khaled*, this phase transformed raw database exports into a machine-learning-ready dataset.
*   **Data Processing:** Cleaned, wrangled, and integrated exactly **22,049 records** spanning across **10 major Turkish cities**.
*   **Feature Engineering:** Engineered sophisticated behavioral features, including Recency, Frequency, and Monetary (RFM) metrics, processing timestamps, and handling outliers/missing values.
*   **Statistical Analysis:** Conducted rigorous univariate and bivariate analysis to uncover hidden correlations in customer purchasing behaviors.

### 🍊 Phase 3: Predictive Modeling & Data Mining (Orange)
Transitioned engineered features into a visual, high-performance machine learning environment.
*   **Pipeline Complexity:** Constructed a massive **35-widget visual programming pipeline**.
*   **Customer Segmentation:** Deployed **K-Means Clustering** to segment the customer base into **3 distinct tiers** (e.g., High-Value, At-Risk, New).
*   **Predictive Analytics:** Trained a highly accurate **Random Forest** classification model dedicated to predicting customer churn and product return likelihood.

### 📊 Phase 4: Business Intelligence & Orchestration (Power BI)
*   **Live Dashboards:** Established live-connection dashboards tracking real-time operational KPIs.
*   **Conversion Tracking:** Successfully monitored the critical **"Cart-to-Order Conversion"** metric, revealing an impressive **85.71%** conversion rate.

## 📈 Key Business Insights

*   🥇 **Revenue Drivers:** **Electronics** conclusively drive the highest gross revenue and profit margins across the catalog.
*   📦 **Volume Drivers:** **Daily Essentials** account for the highest transaction volume and frequency, ensuring consistent cash flow.
*   🗺️ **Geographic Dominance:** **Istanbul** holds absolute dominance in both user base density and total lifetime value compared to the other 9 Turkish cities.
